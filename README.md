# SQL Business Insights

This repository contains SQL business analysis projects completed as part of my SQL Foundation program.

## Repository Structure

- notes/ – Schema documentation and learning notes
- queries/ – SQL solutions for business problems
- INTERPRETATIONS.md – Business insights and observations from each SQL query
- case_study_link.md – Link to the final Notion case study

## Database

All SQL queries are written using the `ecom` schema provided on the internal Metabase server.
